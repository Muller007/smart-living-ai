# Product Requirements Document (PRD)

## Product Name
Smart Living AI

---

## Objective
Help small business owners track finances and make better decisions using WhatsApp.

---

## MVP Features

### 1. Record Income
- User can send income message
- System stores transaction

---

### 2. Record Expense
- User can send expense message
- System stores transaction

---

### 3. Profit Calculation
- System calculates total income, expense, profit

---

### 4. Simple Responses
- Clear and understandable messages

---

## Non-Goals (MVP1)
- AI predictions
- Voice input
- Multi-language support
- Mobile app

---

## Success Criteria
- User can use system without explanation
- System responds correctly every time
- User understands output

---

## Future Features
- AI insights
- Voice support
- Customer tracking
- Financial predictions
